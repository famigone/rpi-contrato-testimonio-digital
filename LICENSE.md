W3C XML Signature Syntax and Processing, publicado por W3C bajo la
[W3C Software License](https://www.w3.org/Consortium/Legal/copyright-software).

## Uso por terceros

Cualquier organización o sistema que quiera integrarse al RPI Neuquén para
enviar testimonios digitales puede usar este contrato como base de su
implementación, sujeto a:

1. Cumplir con los requisitos técnicos descritos en la documentación.
2. Coordinar con el RPI los aspectos operativos (URLs, autenticación, etc.).
3. Respetar la normativa registral aplicable en la Provincia del Neuquén.

El uso del contrato no implica derecho automático a integrarse con los
sistemas productivos del RPI. La habilitación de la integración requiere
acuerdo institucional con el RPI.
